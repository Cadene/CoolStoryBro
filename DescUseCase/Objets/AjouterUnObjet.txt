Description du cas


Identificateur

Nom du cas :  Ajouter un objet

Description :  L'auteur ajoute un objet

Acteurs : 
- Auteur


Séquencement

Hypothèses :
- aucune

Pré-conditions : 
- Un livre est en cours de modification

Scénario Nominal:
1. L'auteur saisie le nom de l'objet à ajouter
2. Le système vérifie que l'objet n'existe pas encore
3. Le système ajoute l'objet à la liste d'objets
4. Le système propose à l'auteur d'ajouter un autre objet
5. L'auteur choisit de finir

Post-conditions :
- L'objet crée est ajouté à la liste d'objet.

Exigences non-fonctionnelles:
-Aucune

Alternatives: 
- SN2: Un objet de ce nom existe déjà : Retour en SN1
- SN5: L'auteur choisit d'ajouter un autre objet: Retour en SN1

Exceptions :
- SN1-3 : l'auteur annule la saisie, fin du cas
