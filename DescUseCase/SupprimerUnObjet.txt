Description du cas


Identificateur

Nom du cas :  Supprimer un objet

Description :  L'auteur supprime un objet.

Acteurs : 
- Auteur


Séquencement

Hypothèses :
- aucune

Pré-conditions : 
- Un livre est en cours de modification

Scénario Nominal:
1. L'auteur saisie le nom de l'objet à supprimer
2. Le système vérifie que l'objet existe
3. Le système supprime l'objet à la liste d'objets
4. Le système propose à l'auteur de supprimer un autre objet
5. L'auteur choisit de finir

Post-conditions :
- L'objet crée est supprimé de la liste d'objet.

Exigences non-fonctionnelles:
-Aucune

Alternatives: 
- SN2: Aucun objet de ce nom n'existe : Retour en SN1
- SN5: L'auteur choisit de supprimer  un autre objet: Retour en SN1

Exceptions :
- SN1-3 : l'auteur annule la saisie, fin du cas
