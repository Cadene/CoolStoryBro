Description du cas


Identificateur

Nom du cas : Ajouter un enchainement

Description :  L'auteur crée un nouvel enchainement

Acteurs : 
- Auteur


Séquencement

Hypohtèses :
- aucune

Pré-conditions : 
- aucune

Scénario Nominal:
1. L'auteur selectionne deux sections entre lesquelles ajouter un enchainement
2. Le système vérifie que la première section existe
3. Le système vérifie que la seconde section existe
4. Le systeme créer l'enchainement
 

Post-conditions :
- Un enchainement a été crée entre les deux sections

Exigences non-fonctionnelles :
- aucune

Alternatives :
-

Exceptions :
- SN1 : L'auteur annule la création
