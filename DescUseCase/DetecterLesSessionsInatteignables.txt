Description du cas


Identificateur

Nom du cas : Visualiser les sections inatteignables

Description :  A partir des graphes des enchainements, affiche les sections inatteignables.

Acteurs : 
- Auteur


Séquencement

Hypohtèses :
- aucune

Pré-conditions : 
- Un livre est en cours de modification

Scénario Nominal:
1. L'auteur confirme l'affichage des sections inatteignables
2. Le système détecte les sections inatteignables du livre
3. Le système affiche les sections inatteignables

Post-conditions :
- Les sections inatteignables sont affichées.

Exigences non-fonctionnelles :


Alternatives :
-Aucune

Exceptions :
-SN-2 : Il n'existe pas de section inatteignable-> Message d'erreur.
