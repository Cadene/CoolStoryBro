Description du cas


Identificateur

Nom du cas :  Ajouter un prérequis

Description :  Ajouter un prérequis

Acteurs : 
- Auteur


Séquencement

Hypohtèses :
- aucune

Pré-conditions : 
- Un enchainement existe entre deux sections

Scénario Nominal:
1. L'auteur selectionne un enchainement existant
2. L'auteur selectionne un objet dans la liste d'objet
3. Le système vérifie que l'objet n'appartient pas déjà à la liste des prérequis
4. Le système ajoute l'objet à la liste des prérequis
5. Le système confirme l'ajout de l'objet et propose à l'auteur d'en rajouter un autre
6. L'auteur choisit de finir.
 

Post-conditions :
- Un prérequis est ajouté pour l'enchainement

Exigences non-fonctionnelles :
- aucune

Alternatives :
- SN3 : L'objet existe déjà dans la liste des prérequis : retour en SN2
- SN6 : L'auteur choisit d'ajouter un autre prérequis : retour en SN2

Exceptions :
- SN1 : L'auteur annule l'ajout de prérequis, retour en SN1
